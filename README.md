# Regulated Data Platform – IFRS9 ECL on Databricks

## Overview
This project implements a regulated-grade credit risk data platform 
supporting IFRS9 Expected Credit Loss calculations using a Lakehouse architecture.

## Tech Stack
- Databricks
- Delta Lake
- Kafka (event ingestion)
- Terraform (Infrastructure as Code)
- GitHub Actions (CI/CD)
- PySpark
